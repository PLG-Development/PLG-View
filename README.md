# PLG View
Einfacher  Bildbetrachter ohne sinnlose Zusatzfunktionen

## Nutzung
Nutzen Sie die Buttons -> und <- um durch die Bilder des geöffneten Ordners zu navigieren.
Bewegen Sie den Zoomslider, um im Bild zu zoomen und bewegen Sie das Bild per Mausdrag.

Unter [...] finden Sie weitere Optionen:
	-	Datei öffnen -> Öffnen eines Bildes
	-	Ordner öffnen -> Öffnen eines Ordners
	-	Modus ändern-> Ändern der Ansicht (siehe Ansichten)
	-	Vollbild aktivieren/deaktivieren

## Ansichten
### Standardmodus
Die Steuerelemente sind in der Mitte am unteren Rand, das Menü auf der rechten Seite.

### Tafelmodus
Steuerelemente sind auf der rechten und linken Seite am unteren Rand dupliziert vorzufinden, das Menü befindet sich in der Mitte.

### Konzentrationsmodus
Es sind keine Steuerelemente sichtbar.

## Tastenkombinationen

| Tastenkombination | Befehl |
| ------------------|--------|
| Strg + F | Ordner öffnen |
| Strg + O | Datei öffnen |
| Strg + T | Tafelmodus |
| Strg + S | Standardmodus |
| Strg + V | Konzentrationsmodus |
| Strg + + | Reinzoomen |
| Strg + - | Rauszoomen |
| F11 | Vollbild |
| Pfeil nach rechts | Nächstes Bild |
| Pfeil nach links | Vorheriges Bild |
| Rechtsklick | Menü öffnen |
